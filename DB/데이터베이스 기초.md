# [1] DB 기초
## 1. maria DB
### 1. 데이터베이스 생성
  
- 설치 경로 
  - mysql -u root -p 

- 데이터베이스 생성
  - CREATE DATABASE {DB_NAME} CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
- 데이터베이스 사용
  - use study 
- 사용자 생성
  - CREATE USER {USER_NAME}@{HOST_NAME} IDENTIFIED BY ‘{PASSWORD}’;
  - HOST_NAME : 
  아이피 - 해당 아이피에서만 접속 가능  
  Localhost - 로컬에서만 접속가능  
  % - 외부에서 접속가능


### etc.

- '_' 문자열 하나 'Mar_ %'
