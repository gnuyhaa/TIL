# [1] git 기초
git : 소프트웨어  
저장소 : git으로 관리되고 있는 애  
로컬 저장소 : 내 컴퓨터에 있는 저장소  
원격 저장소 : github에 있는 저장소

git 설치 후 해야할 일  
git config 설정( user.name, user.email)

파일 상태 확인
- git status

저장소 선언 
- git init

1. git add . # 추가된 모든 파일이 추가됨
2. git commit -m '커밋 메시지'
3. git push origin main

파일의 4가지 상태
- Untracked : 
- Unmodified : 
- Modified :
- Staged :
